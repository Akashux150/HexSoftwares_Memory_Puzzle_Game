# 🧩 Memory Puzzle Game

## 📌 Project Overview

This project is a **Memory Puzzle Game** developed using Python and Tkinter.
The objective of the game is to match pairs of cards within the shortest possible time.

---

## 🚀 Features

* 🧩 4x4 interactive card grid
* 🔢 Cards reveal numbers on click
* ✅ Matching logic for pairs
* ❌ Cards flip back if not matched
* ⏱️ Real-time timer
* 🎉 Win message with completion time

---

## 🛠️ Technologies Used

* Python 🐍
* Tkinter (GUI Library)
* Random Module
* Time Module


---

## ▶️ How to Play

1. Click on any card to reveal its value
2. Click on another card to find its pair
3. If both cards match → they stay visible ✅
4. If not → they flip back ❌
5. Match all pairs to win the game 🎉

---

## 📸 Output

* Interactive GUI window
* Timer displayed at the top
* Win message after completing the game
